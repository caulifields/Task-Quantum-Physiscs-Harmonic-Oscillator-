Soal: Tentukan bentuk eksplisit dari fungsi gelombang untuk sistem Osilator 
Harmonik pada tingkatan tenaga ke n = 9. Pada tingkatan tenaga tersebut, 
dimanakah partikel paling boleh jadi ditemukan?  
Langkah-langkah pengerjaan : 
a. Bahasa Pemrograman 
Program dibuat menggunakan bahasa C++ dengan library standar cmath, 
iostream, dan fstream. 
b. Langkah-langkah Program 
1. 
Mendefinisikan konstanta fisika m=ω=ℏ=1m = \omega = \hbar = 
1m=ω=ℏ=1 untuk penyederhanaan. 
2. Membuat fungsi Hermite Polynomial secara rekursif. 
3. Menghitung fungsi gelombang ψ (x) untuk n = 9. 
4. Menghitung kebolehjadian ∣ψn(x)∣2|ψ (x)|^2∣ψn (x)∣2. 
5. Menyimpan hasil ke dalam file hasil_osilator.csv untuk dianalisis atau 
digrafikkan.
